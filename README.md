# GraphicViewer
**One app for all your media — by Logic Dynamics**

A Microsoft Photos-inspired media suite built with Blazor WebAssembly + .NET 10.

## Features
- 🔷 SVG Viewer — browse and display SVG files
- 🖼 PNG/JPEG Viewer — view images with download support
- 🎬 MP4 Player — play videos with native controls
- 🎵 MP3 Player — full audio player with shuffle, repeat, visualizer
- 📺 YouTube Player — embed videos, save playlist to localStorage
- 🏠 Home Dashboard — quick-launch all viewers

## Tech Stack
Blazor WASM · .NET 10 · C# · JSInterop
